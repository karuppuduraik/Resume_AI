# 📖 Overview

Resume AI Builder is a full-stack MERN application that allows users to create ATS-friendly resumes with the assistance of Google Gemini AI.

The application provides intelligent content generation, resume management, Google authentication, premium membership support, and an admin dashboard for user management.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/karuppuduraik/Resume_AI.git
```

```
cd Resume_AI
```

---

## Backend

```
cd server
npm install
npm run dev
```

---

## Frontend

```
cd client
npm install
npm run dev
```

---

# ⚙ Environment Variables

### Server (.env)

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET

GEMINI_API_KEY=YOUR_API_KEY

GOOGLE_CLIENT_ID=YOUR_CLIENT_ID

ADMIN_EMAILS=YOUR_EMAIL
```

---

### Client (.env)

```env
VITE_API_URL=http://localhost:5000/api

VITE_GOOGLE_CLIENT_ID=YOUR_GOOGLE_CLIENT_ID
```
